# GoodbyeDPI Türkiye

Bu site, Türkiye'deki internet kısıtlamalarını ve yasaklarını aşmak için kullanılan **GoodbyeDPI** aracını hızlıca indirmenizi sağlar.

## Nasıl Kullanılır?

1.  Yukarıdaki **"Aracı İndir"** butonuna tıklayarak ZIP dosyasını indirin.
2.  ZIP dosyasını masaüstüne klasör olarak çıkartın.
3.  Klasörün içindeki `1_idp_list.cmd` veya benzeri bir `.cmd` dosyasını çalıştırın.
4.  İnternet kısıtlamaları kalkacaktır.
